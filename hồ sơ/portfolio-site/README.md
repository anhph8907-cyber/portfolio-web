# Modern Personal Portfolio Site

A complete, modern, editable personal portfolio website.

## Features
- Minimalist, high-end UI inspired by Apple/Notion/Framer
- Full responsive layout (desktop/tablet/mobile)
- Smooth animations, hover effects, reveal on scroll
- Sticky navbar + smooth scrolling
- Dark mode toggle (persist via localStorage)
- Content editable mode for inline updates (persist via localStorage)
- Project modal details and contact form
- SEO-friendly sections and structure
- Page transitions + micro-interactions

## Getting Started
1. Open the folder in VS Code.
2. Launch using Live Server extension, or open `index.html` in browser.

## Editable Content
- Click `Edit mode` button in navbar to enable inline editing on texts.
- Your edits persist locally in browser storage.
- Click `Save mode` to lock editing.

## Deploy (GitHub Pages)
1. Initialize git:
   - `git init`
   - `git add .`
   - `git commit -m "Initial portfolio site"
2. Push to GitHub repository.
3. In repo settings -> Pages -> set branch to `main` (or `gh-pages`) and folder to `/root`.
4. Public link will be presented by GitHub.

## Optional Hosting (Vercel / Netlify)
- Connect repository and deploy with a single click.

## Notes
- Project images are from unsplash placeholders but can be replaced with your own.
- Customize colors, typography, content in `styles.css` / `index.html`.
